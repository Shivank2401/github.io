# GitHub Pages Setup

1. Create a GitHub repo and push this project (including `docs/`).
2. Repo → **Settings** → **Pages**.
3. **Source:** Deploy from a branch.
4. **Branch:** main (or default).
5. **Folder:** `/docs`.
6. Save. Site URL: `https://<username>.github.io/<repo>/` or `https://<username>.github.io/` if repo is `username.github.io`.
7. `docs/.nojekyll` is present so GitHub doesn’t run Jekyll.
