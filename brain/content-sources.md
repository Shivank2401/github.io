# Content Sources for Portfolio

- **word.py** → Generates `Shivank_Rajput_ATS_CV.docx`: full CV (summary, skills, experience, projects, education, certifications). Used as primary source for site copy.
- **ppt.py** → Generates `Shivank_Rajput_Portfolio_Updated.pptx`: title slide, executive profile, competency matrix, enterprise ecosystem, automation workflow, Power BI portfolio, contact. Skills and structure aligned with DOCX.
- **PDF:** User may have a PDF resume; currently no PDF extraction script in project. Site content is aligned with DOCX/PPTX. If a PDF is added later, a script can be added to extract text and optionally sync to the site or to a data file.
